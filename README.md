# AWS-SAA_C03

## Analytics
- **Amazon Athena**: An interactive query service that makes it easy to analyze data directly in Amazon S3 using standard SQL.
- **AWS Data Exchange**: A managed service for cataloging, cleaning, and transforming data, often used to prepare data for analytics and machine learning.
- **AWS Data Pipeline**: A web service that helps you reliably process and move data between different AWS compute and storage services, as well as on-premises data sources, at specified intervals.
- **Amazon EMR**: A cloud big data platform that simplifies the process of running big data frameworks like Apache Spark and Hadoop.
- **AWS Glue**: A fully managed extract, transform, and load (ETL) service that makes it easy to prepare and load your data for analytics. It includes features like a data catalog, automatic schema discovery, and code generation.
- **Amazon Kinesis**: A real-time data streaming service that makes it easy to collect, process, and analyze streaming data. It consists of multiple services, including Kinesis Data Streams, Kinesis Data Firehose, Kinesis Data Analytics, and Kinesis Video Streams.
- **AWS Lake Formation**: A managed service that makes it easy to set up, secure, and manage data lakes. It simplifies tasks like data ingestion, cleaning, cataloging, and permission management.
- **Amazon Managed Streaming for Apache Kafka (Amazon MSK)**: A fully managed Apache Kafka service that makes it easy to ingest and process streaming data in real-time.
- **Amazon OpenSearch Service (Amazon Elasticsearch Service)**: A fully managed service for deploying, securing, and operating OpenSearch clusters at scale.
- **Amazon QuickSight**: A fully managed service that provides interactive visualization and machine learning capabilities for business intelligence.
- **Amazon Redshift**: A fully managed, petabyte-scale cloud data warehouse service that makes it simple and cost-effective to analyze all your data using standard SQL and your existing business intelligence tools.

## Application Integration
- **Amazon AppFlow**: A fully managed integration service that enables you to securely transfer data between Software-as-a-Service (SaaS) applications like Salesforce, Zendesk, Slack, and ServiceNow, and AWS services like Amazon S3 and Amazon Redshift, in just a few clicks.
- **AWS AppSync**: A serverless GraphQL and Pub/Sub API service that simplifies building modern web and mobile applications.
- **Amazon EventBridge (Amazon CloudWatch Events)**: A serverless event bus that makes it easier to build event-driven applications at scale using events generated from your applications, integrated Software-as-a-Service (SaaS) applications, and AWS services.
- **Amazon MQ**: A managed message broker service for Apache ActiveMQ and RabbitMQ, making it easy to set up and operate message brokers in the cloud.
- **Amazon Simple Notification Service (Amazon SNS)**: A highly available, durable, and secure pub/sub messaging service that enables you to decouple microservices, distributed systems, and serverless applications.
- **Amazon Simple Queue Service (Amazon SQS)**: A highly scalable, durable, and secure queue service that lets you decouple and scale microservices, distributed systems, and serverless applications.
- **AWS Step Functions**: A serverless function orchestrator that makes it easy to sequence multiple AWS Lambda functions and other services into flexible workflows.

## AWS Cost Management
- **AWS Budgets**: A service that allows you to set custom budgets to track your cost and usage and receive alerts when your costs or usage exceed (or are forecasted to exceed) your budgeted amount.
- **AWS Cost and Usage Report**: A comprehensive report that contains detailed information about your AWS costs and usage, which can be configured to provide hourly, daily, or monthly data at the resource level. It is the most detailed source of cost and usage data available.
- **AWS Cost Explorer**: A web service that makes it easy to visualize, understand, and manage your AWS costs and usage over time. It allows you to analyze your spending trends and identify areas for optimization.
- **Savings Plans**: A cost management service that offers flexible, discounted pricing models in exchange for a commitment to a consistent amount of compute usage (measured in USD/hour) over a one or three-year term.

## Compute
- **AWS Batch**: A fully managed service that allows you to easily and efficiently run your containerized jobs (like batch processing or machine learning inference) at any scale, without needing to provision, manage, or scale compute infrastructure.
- **Amazon EC2**: A web service that provides secure and resizable compute capacity in the cloud as virtual machines (instances).
- **Amazon EC2 Auto Scaling**: A service that automatically adjusts the desired number of Amazon EC2 instances in your Auto Scaling group according to fluctuating demand.
- **AWS Elastic Beanstalk**: A service for deploying and scaling web applications and services developed with popular languages like Java, .NET, PHP, Node.js, Python, Ruby, and Go.
- **AWS Outposts**: A fully managed service that extends AWS infrastructure, services, APIs, and tools to virtually any customer datacenter, co-location space, or on-premises facility for a truly consistent hybrid experience.
- **AWS Serverless Application Repository**: A managed repository for serverless applications. It allows developers to quickly deploy pre-built serverless architectures and share their own.
- **VMware Cloud on AWS**: A service that allows you to run VMware vSphere-based workloads in the AWS Cloud.
- **AWS Wavelength**: A specialized AWS infrastructure service that provides mobile edge computing infrastructure for developing, deploying, and scaling ultra-low latency applications to mobile devices.

## Containers
- **Amazon Elastic Container Registry (Amazon ECR)**: A fully managed Docker container registry that makes it easy for developers to store, manage, and deploy Docker container images.
- **Amazon Elastic Container Service (Amazon ECS)**: A fully managed container orchestration service that makes it easy to run, stop, and manage Docker containers on a cluster of Amazon EC2 instances or on-premises using ECS Anywhere.
- **Amazon ECS Anywhere**: A capability in Amazon Elastic Container Service (ECS) that allows you to run ECS tasks and services on customer-managed infrastructure.
- **Amazon Elastic Kubernetes Service (Amazon EKS)**: A fully managed Kubernetes service that makes it easy to run Kubernetes on AWS without needing to install, operate, and maintain your own Kubernetes control plane.
- **Amazon EKS Anywhere**: A distribution of Kubernetes that you can use to create and manage highly available Kubernetes clusters on your own infrastructure.
- **Amazon EKS Distro**: A managed Kubernetes distribution that can be used to create, manage, and operate Kubernetes clusters on your own infrastructure or on public clouds.

## Database
- **Amazon Aurora**: A relational database service that combines the speed and availability of high-end commercial databases with the simplicity and cost-effectiveness of open-source databases.
- **Amazon Aurora Serverless**: A serverless, on-demand, auto-scaling configuration for Amazon Aurora.
- **Amazon DocumentDB (with MongoDB compatibility)**: A fully managed NoSQL document database service that is compatible with MongoDB APIs.
- **Amazon DynamoDB**: A fully managed NoSQL key-value and document database that delivers single-digit millisecond performance at any scale.
- **Amazon ElastiCache**: A fully managed in-memory data store service that supports caching and real-time use cases (Redis and Memcached).
- **Amazon Keyspaces (for Apache Cassandra)**: A fully managed Apache Cassandra-compatible database service.
- **Amazon Neptune**: A fully managed graph database service.
- **Amazon Quantum Ledger Database (Amazon QLDB)**: A fully managed ledger database that provides a transparent, immutable, and cryptographically verifiable transaction log.
- **Amazon RDS**: A fully managed relational database service that makes it easy to set up, operate, and scale a relational database in the cloud.
- **Amazon Timestream**: A fully managed, fast, scalable, and serverless time series database service for IoT and operational applications.

## Developer Tools
- **AWS X-Ray**: A service that makes it easy to debug, analyze, and optimize the performance of your distributed applications.

## Front-End Web and Mobile
- **AWS Amplify**: A fully managed service for building, deploying, and hosting modern web and mobile applications.
- **Amazon API Gateway**: A fully managed service that makes it easy for developers to create, publish, maintain, monitor, and secure APIs at any scale.
- **AWS Device Farm**: A service that helps you manage mobile devices and web browsers for testing applications.
- **Amazon Pinpoint**: A multi-channel marketing communication service that helps you engage with your customers across various channels.

## Machine Learning
- **Amazon Comprehend**: A machine learning service that uses natural language processing (NLP) to extract insights and relationships from unstructured text.
- **Amazon Forecast**: A fully managed service that uses machine learning to deliver highly accurate forecasts.
- **Amazon Fraud Detector**: A fully managed service that uses machine learning to identify potentially fraudulent online activities.
- **Amazon Kendra**: A fully managed intelligent search service powered by machine learning.
- **Amazon Lex**: A service for building conversational interfaces (chatbots, voice bots) into any application.
- **Amazon Polly**: A machine learning service that turns text into lifelike speech.
- **Amazon Rekognition**: A fully managed service that provides image and video analysis capabilities using deep learning.
- **Amazon SageMaker**: A fully managed service for developers, data scientists, and business analysts to quickly build, train, and deploy machine learning models at scale.
- **Amazon Textract**: A machine learning service that automatically extracts text and data from virtually any document.
- **Amazon Transcribe**: A machine learning service that converts speech to text.
- **Amazon Translate**: A neural machine translation service that delivers fast, high-quality, and affordable language translation.

## Management and Governance
- **AWS Auto Scaling**: Monitors your applications and automatically adjusts capacity to maintain steady, predictable performance at the lowest possible cost.
- **AWS CloudFormation**: A service that gives developers and businesses an easy way to create a collection of related AWS and third-party resources, and provision and manage them in an orderly and predictable fashion.
- **AWS CloudTrail**: A service that enables governance, compliance, operational auditing, and risk auditing of your AWS account.
- **Amazon CloudWatch**: A monitoring and observability service for DevOps engineers, developers, site reliability engineers (SREs), and IT managers.
- **AWS Command Line Interface (AWS CLI)**: A unified tool to manage your AWS services.
- **AWS Compute Optimizer**: Recommends optimal AWS compute resources for your workloads to reduce costs and improve performance.
- **AWS Config**: A service that enables you to assess, audit, and evaluate the configurations of your AWS resources.
- **AWS Control Tower**: The easiest way to set up and govern a secure, multi-account AWS environment.
- **AWS License Manager**: A service that makes it easier to manage your software licenses from software vendors such as Microsoft, SAP, Oracle, and IBM across AWS and on-premises environments.
- **Amazon Managed Grafana**: A fully managed and secure data visualization service.
- **Amazon Managed Service for Prometheus**: A fully managed Prometheus-compatible monitoring service.
- **AWS Management Console**: A web-based interface for accessing and managing AWS services.
- **AWS Organizations**: A service that helps you centrally manage and automate environments across multiple AWS accounts.
- **AWS Personal Health Dashboard**: Provides alerts and guidance when AWS is experiencing events that may impact you.
- **AWS Proton**: The first fully managed delivery service for container and serverless applications.
- **AWS Service Catalog**: Allows organizations to create and manage catalogs of IT services that are approved for use on AWS.
- **AWS Systems Manager**: A secure end-to-end management solution for your hybrid cloud environment.
- **AWS Trusted Advisor**: An online tool that provides you real-time guidance to help you provision your resources following AWS best practices.
- **AWS Well-Architected Tool**: A tool to help you review the state of your workloads and compares them to the latest AWS architectural best practices.

## Media Services
- **Amazon Elastic Transcoder**: A media transcoding in the cloud.
- **Amazon Kinesis Video Streams**: A fully managed AWS media service that streams live video from devices to the AWS Cloud.

## Migration and Transfer
- **AWS Application Discovery Service**: Helps enterprise customers plan migration projects by gathering information about their on-premises data centers.
- **AWS Application Migration Service (CloudEndure Migration)**: The primary migration service for lift-and-shift migrations of applications to AWS.
- **AWS Database Migration Service (AWS DMS)**: A service that supports homogenous and heterogeneous migrations between different database platforms.
- **AWS DataSync**: A robust data transfer service that makes it easy to automate moving data between on-premises storage and Amazon S3 or Amazon EFS.
- **AWS Migration Hub**: A single location to track the progress of application migrations across multiple AWS and partner solutions.
- **AWS Server Migration Service (AWS SMS)**: An agentless service which makes it easier and faster for you to migrate thousands of on-premises workloads to AWS.
- **AWS Snow Family**: A collection of physical devices that help migrate large amounts of data into and out of the AWS Cloud without moving it over the internet.
- **AWS Transfer Family**: A fully managed support for file transfers directly into and out of Amazon S3 using SFTP, FTPS, and FTP.

## Networking and Content Delivery
- **Amazon CloudFront**: A fast content delivery network (CDN) service that securely delivers data, videos, applications, and APIs to customers globally.
- **AWS Direct Connect**: A cloud service solution that makes it easy to establish a dedicated network connection from your premises to AWS.
- **Elastic Load Balancing (ELB)**: Automatically distributes incoming application traffic across multiple targets, such as Amazon EC2 instances, containers, IP addresses, and Lambda functions.
- **AWS Global Accelerator**: A networking service that improves the performance of your users' traffic by up to 60% using the AWS global network infrastructure.
- **AWS PrivateLink**: Provides private connectivity between VPCs, AWS services, and your on-premises networks.
- **Amazon Route 53**: A highly available and scalable cloud Domain Name System (DNS) web service.
- **AWS Transit Gateway**: A service that connects VPCs and on-premises networks through a central hub.
- **Amazon VPC**: A service that lets you launch AWS resources into a logically isolated virtual network that you define.
- **AWS VPN**: Solutions to establish secure connections between your on-premises networks, remote offices, client devices, and the AWS global network.
- **NAT Gateway**: A highly available, managed Network Address Translation (NAT) service for your resources in a private subnet to access the internet.

## Security, Identity, and Compliance
- **AWS Artifact**: Your go-to, central resource for compliance-related information that matters to you.
- **AWS Audit Manager**: Helps you continuously audit your AWS usage to simplify how you assess risk and compliance.
- **AWS Certificate Manager (ACM)**: A service that lets you easily provision, manage, and deploy public and private Secure Sockets Layer/Transport Layer Security (SSL/TLS) certificates.
- **AWS CloudHSM**: A cloud-based hardware security module (HSM) that enables you to easily generate and use your own encryption keys on the AWS Cloud.
- **Amazon Cognito**: A service that provides authentication, authorization, and user management for your web and mobile apps.
- **Amazon Detective**: A service that makes it easy to analyze, investigate, and identify the root cause of potential security issues or suspicious activities.
- **AWS Directory Service**: A managed service that allows you to connect your AWS resources with an existing on-premises Microsoft Active Directory or to set up a new, stand-alone directory in the AWS Cloud.
- **AWS Firewall Manager**: A security management service which allows you to centrally configure and manage firewall rules across your accounts and applications in AWS Organizations.
- **Amazon GuardDuty**: A threat detection service that continuously monitors for malicious activity and unauthorized behavior to protect your AWS accounts and workloads.
- **AWS Identity and Access Management (IAM)**: A web service that helps you securely control access to AWS resources.
- **Amazon Inspector**: an automated security assessment service that helps improve the security and compliance of applications deployed on AWS.
- **AWS Key Management Service (AWS KMS)**: A managed service that makes it easy for you to create and control the cryptographic keys used to encrypt your data.
- **Amazon Macie**: A fully managed data security and data privacy service that uses machine learning and pattern matching to discover and protect your sensitive data in AWS.
- **AWS Network Firewall**: A managed service that makes it easy to deploy essential network protections for all of your Amazon Virtual Private Clouds (VPCs).
- **AWS Resource Access Manager (AWS RAM)**: A service that enables you to easily and securely share AWS resources with any AWS account or within your AWS Organization.
- **AWS Secrets Manager**: A service that helps you protect secrets needed to access your applications, services, and IT resources.
- **AWS Security Hub**: A cloud security posture management service that performs security best practice checks, aggregates alerts, and enables automated remediation.
- **AWS Shield**: A managed Distributed Denial of Service (DDoS) protection service that safeguards applications running on AWS.
- **AWS Single Sign-On (AWS IAM Identity Center)**: A cloud SSO service that makes it easy to centrally manage SSO access to all of your AWS accounts and cloud applications.
- **AWS WAF**: A web application firewall that helps protect your web applications or APIs against common web exploits and bots.

## Serverless
- **AWS AppSync**: (See Application Integration)
- **AWS Fargate**: A serverless compute engine for containers that works with both Amazon Elastic Container Service (ECS) and Amazon Elastic Kubernetes Service (EKS).
- **AWS Lambda**: A serverless compute service that lets you run code without provisioning or managing servers, creating workload-aware cluster scaling logic, maintaining event integrations, or managing runtimes.

## Storage
- **AWS Backup**: A fully managed backup service that makes it easy to centralize and automate the back up of data across AWS services.
- **Amazon Elastic Block Store (Amazon EBS)**: An easy to use, high-performance, block-storage service designed for use with Amazon Elastic Compute Cloud (Amazon EC2).
- **Amazon Elastic File System (Amazon EFS)**: A simple, serverless, set-and-forget, elastic file system that lets you create and configure file systems quickly.
- **Amazon FSx**: A service that makes it easy to launch and run popular file systems (Windows File Server, Lustre, OpenZFS, NetApp ONTAP) that are fully managed.
- **Amazon S3**: A service that offers industry-leading scalability, data availability, security, and performance for object storage.
- **Amazon S3 Glacier**: A secure, durable, and extremely low-cost cloud storage class for data archiving and long-term backup.
- **AWS Storage Gateway**: A hybrid cloud storage service that gives you on-premises access to virtually unlimited cloud storage.
