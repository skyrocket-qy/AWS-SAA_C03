# Comparison: Amazon EKS vs. EKS Anywhere vs. EKS Distro

| Feature | **Amazon EKS** | **Amazon EKS Anywhere** | **Amazon EKS Distro** |
| :--- | :--- | :--- | :--- |
| **Description** | Fully managed Kubernetes service on AWS. | Run EKS on your own infrastructure (on-premises). | The open-source distribution of Kubernetes used by EKS. |
| **Managed By** | AWS (Control Plane) | You (with AWS automation tools) | You (Manual) |
| **Location** | AWS Cloud | On-Premises / Hybrid | Anywhere (On-Prem / Cloud) |
| **Control Plane** | Managed by AWS | Running on your infrastructure | Managed by you |
| **Cost** | Hourly per cluster + resources | Open source (optional support subscription) | Free (Open Source) |
